Indian Regulatory Compliance

✓
Digital Personal Data Protection (DPDP) Act, 2023India's primary data protection law. Wapas processes only the minimum patient data required for treatment follow-up. Every patient message includes a STOP opt-out, honouring the right to withdraw consent as required under Section 6 of the DPDP Act.
✓
IT (Reasonable Security Practices) Rules, 2011Patient records qualify as Sensitive Personal Data under Rule 3. Wapas maintains per-clinic data silos — no patient from one clinic is ever visible to another — and enforces strict access controls at every layer.
✓
TCCCP Regulations (Telecom)All patient communications are initiated only after dentist approval and only to patients with an existing clinical relationship with the clinic. Every message clearly identifies the sending clinic.
✓
WhatsApp Business API PolicyAll outbound patient messages use pre-approved WhatsApp Business API templates. Opt-out is honoured immediately. No unsolicited bulk messaging.
🔐

Explicit approval before every message

The dentist reviews and approves every patient outreach. Wapas never sends anything without a direct YES from the clinic owner. This is not just a feature — it is the privacy architecture made visible.

🏥

Complete data isolation between clinics

Dr. Sharma's patient records are never visible to any other Wapas account — ever. Each clinic is a fully isolated data environment. Shared infrastructure, zero data sharing.

📵

STOP opt-out in every message

Every patient message includes a clear opt-out instruction. Replies of STOP are processed immediately and that patient is never contacted again — compliant with DPDP Act 2023 consent withdrawal requirements.

📋

Minimum data principle

Wapas reads only four fields — name, phone number, procedure type, and last visit date. No billing data, no medical notes, no insurance information. We take only what is needed.

🎯

Purpose limitation

Patient data collected via Wapas is used exclusively for treatment follow-up communication with that clinic's own patients. Never used for marketing, analytics, or any secondary purpose.
